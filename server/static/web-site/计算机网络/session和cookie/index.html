    # session 和 cookie

    • session 代表一次会话
    • session 是这次会话的唯一标识

    cookie 是浏览器存储的手段

    提示

    用户登陆后，服务器会先进行登陆校验，登陆校验成功后，会生成一个 sessionid 返回给前端。前端把这个存到 cookie 里
